For my final project I decided to implement the Depth of Field rendering technique. The reason why I chose this was because it creates an image that is more realistic and looks like a focused picture.

To implement the Depth of Field, you must follow these steps:

  1. Cast a ray, and find the point on the ray with the given focal length.
  2. Select a random origin on the aperture
  3. Find the direction vector from the origin to the calculated point
  4. Use the new origin and direction to create a new ray
  5. Repeat the steps 128 times. With the final color divide it by 128
